Getting Started

Press Cmd+G / Ctrl+G to capture a problem from your screen. The solution will appear in the overlay window.

The overlay starts in click-through mode by default, allowing you to type and interact with applications behind it. Press Cmd+T / Ctrl+T to toggle this behavior when you need to interact with the overlay.

If you're sharing your screen, press Cmd+U / Ctrl+U to activate ultra-stealth mode which makes the overlay completely invisible to screen capture tools.
